HTML Tables Cheatsheet

Table Tags

Tags Name Description
<table> Table Element The table element has content that is used to represent a two-dimensional table made of rows and columns.
<tbody> Table Body Is a semantic element that will contain all table data other than table heading and table footer content.
<thead> Table Head Is used to group header content. Is used in conjuction with the <tbody> and <tfoot> elements. Is usually in the form of column headers
<tr> Table row Is used to add rows to a table before adding table data and table headings.
<td> Table Data Can be nested inside a table row element to add a cell of data to a table.
<th> Table Heading Is used to add titles to rows and columns of a table and must be enclosed in a table row element
<tfoot> Table Footer uses table rows to give footer content or to summarize content at the end of a table

Table Attributes

Tags Name Description
rowspan Row Span Indicates how many columns that particular cell should span within the table. The colspan value is set to 1 by default and will take any positive integer between 1 and 1000.
colspan Column Span Indicates how many rows that particular cell should span within the table. The rowspan value is set to 1 by default and will take any positive integer up to 65534.
